# Pasaporte Negro - Motor de documentos

Motor de generacion de dossiers y vouchers. Tres modulos:
- pnpdf.py: motor base (roles tipograficos, registro de cajas, isotipo vectorial)
- pnart.py: telar guilloche y grabados de destino
- dossier_factory.py: fabrica que genera el dossier de cualquier expedicion desde su public_content

Versionado desde la sesion de trabajo.
